1. npm install jasmine-core(mocha if using mocha) karma karma-jasmine(karma-mocha is using mocha)
2. npm install karma-cli -g (karma.exe)
3. Create the test cases and suites
4. karma init (karma config file) --> karma.config.jasmine
5. karma start - to start the running of the tests
6. To debug, Go to chrome -> Debug -> Sources -> base -> Test file -> Put the break points
